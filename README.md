# AI Agent Team GUI (Starter)

This is a starter project for running a small **team of AI agents** (web/search, finance, translator) behind a simple **Streamlit** UI.
It's written so you can plug in the `phi` agent library (or any similar one) and swap the provider to OpenAI / Groq.

⚠️ This repo does **not** include your real API keys. Fill in `config.example.toml` → copy to `config.toml` locally.
